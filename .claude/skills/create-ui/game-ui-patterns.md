# ゲームUI設計パターン

2Dアクションゲーム（サイドスクロール）向けのUI設計パターン集。
画面種別ごとのレイアウト原則、ゲーム固有の要件をまとめる。

---

## 1. HUD（ヘッドアップディスプレイ）

### 原則
- **ゲーム画面を遮らない**: 画面端に配置、半透明背景
- **視認性**: 背景が変化しても読めるよう `text-shadow` とコントラスト確保
- **情報量の制御**: 常時表示は最小限、状況に応じて表示/非表示を切替
- **視線の流れ**: プレイヤーキャラの近くに重要情報（HP等）、補助情報は端に

### レイアウト慣習
```
┌──────────────────────────────────────────────────┐
│ [プレイヤーHP/MP]                    [ミニマップ] │
│ [仲間HP]                                         │
│                                                  │
│                                                  │
│                  (ゲーム画面)                      │
│                                                  │
│                [ボスHP（出現時のみ）]               │
│                                                  │
│ [スキルアイコン]           [通貨] [アイテムスロット] │
└──────────────────────────────────────────────────┘
```

### 実装パターン
- ルート: `position: absolute; width: 100%; height: 100%`
- 各パネル: `position: absolute` で四隅/辺に固定
- バー: `bar-container` (背景) + `bar-fill` (前景、width%で増減)
- ボスHP: `display: none` → C#から `display: flex` に切替
- ダメージ数字: WorldSpace → ScreenSpace変換してUI上にオーバーレイ

### このプロジェクトの既存パターン
- `HUD.uxml` / `HUD.uss` → `HudController.cs` の構成を踏襲
- バーアニメーションは LitMotion を使用
- テキスト更新は値変更時のみ（毎フレーム文字列生成しない）

## 2. メニュー（タイトル/ポーズ/ゲームオーバー）

### 原則
- **フォーカスナビゲーション必須**: パッド操作時に上下移動で選択
- **初期フォーカス**: 画面表示時に最初のボタンに自動フォーカス
- **遷移アニメーション**: フェードイン/アウトで画面切替
- **TimeScale制御**: ポーズメニューは `Time.timeScale = 0` と連動

### レイアウト慣習
```
┌──────────────────────────────┐
│                              │
│        [タイトル]             │
│                              │
│      [ ゲーム開始  ]          │
│      [ 設定       ]          │
│      [ 終了       ]          │
│                              │
│                 [バージョン]  │
└──────────────────────────────┘
```

### MenuStackManager 連携
- メニュー画面は `MenuStackManager.Push()` / `Pop()` で管理
- ESCキーで `Pop()` → 前の画面に戻る
- 設定→音量のような深い階層もスタックで自然にハンドリング

### 実装パターン
```css
/* オーバーレイ背景 */
.menu-overlay {
    position: absolute;
    width: 100%;
    height: 100%;
    background-color: rgba(0, 0, 0, 0.7);
    align-items: center;
    justify-content: center;
}

/* メニューコンテナ */
.menu-container {
    flex-direction: column;
    align-items: center;
    padding: 40px 60px;
    background-color: rgba(20, 20, 30, 0.9);
    border-radius: 8px;
}

/* メニューボタン */
.menu-button {
    width: 240px;
    height: 48px;
    margin: 6px 0;
    font-size: 18px;
    -unity-text-align: middle-center;
    background-color: rgba(60, 60, 80, 0.8);
    border-radius: 4px;
    border-width: 0;
    color: rgb(220, 220, 230);
    transition-property: scale, background-color;
    transition-duration: 0.15s;
}

.menu-button:hover {
    background-color: rgba(80, 80, 110, 0.9);
    scale: 1.03 1.03;
}

.menu-button:focus {
    border-width: 2px;
    border-color: var(--color-accent);
}
```

## 3. ダイアログ（会話/確認）

### 原則
- **会話ウィンドウ**: 画面下部に横長、キャラ立ち絵は左側
- **確認ダイアログ**: 画面中央モーダル、背景は暗転
- **テキスト送り**: クリック/決定ボタンで次の文へ進む
- **選択肢**: 上下選択可能、選択中の項目をハイライト

### レイアウト慣習（会話）
```
┌──────────────────────────────────────────┐
│                (ゲーム画面)                │
│                                          │
├──────────────────────────────────────────┤
│ ┌────┐                                  │
│ │立絵│ [キャラ名]                        │
│ │    │ テキストがここに表示される。         │
│ └────┘ テキスト送りアイコン ▼             │
├──────────────────────────────────────────┤
│   → 選択肢1                              │
│     選択肢2                              │
│     選択肢3                              │
└──────────────────────────────────────────┘
```

### レイアウト慣習（確認ダイアログ）
```
┌──────────────────────────────┐
│      暗転オーバーレイ         │
│   ┌────────────────────┐     │
│   │  タイトルに戻りますか？│     │
│   │                    │     │
│   │  [はい]   [いいえ]  │     │
│   └────────────────────┘     │
└──────────────────────────────┘
```

### 破壊的操作の確認
- セーブデータ削除、タイトルに戻る等は **必ず確認ダイアログ** を経由
- 確認ダイアログのデフォルトフォーカスは **「いいえ」（安全な方）** に設定
- 確認ボタンは **危険色**（赤系）で視覚的に警告

## 4. インベントリ / リスト系

### 原則
- **グリッドレイアウト**: Flexbox wrap で擬似グリッド
- **選択状態**: 選択中アイテムはボーダー or 背景色で示す
- **詳細パネル**: 選択アイテムの詳細を右側/下部に表示
- **空の状態**: アイテムがない時は次のアクションを明示

### レイアウト慣習
```
┌────────────────────────────────────┐
│ [カテゴリタブ: 全て | 武器 | 防具]   │
├──────────────────┬─────────────────┤
│ ┌──┐ ┌──┐ ┌──┐ │ [アイテム名]     │
│ │  │ │  │ │  │ │ [アイテム画像]   │
│ └──┘ └──┘ └──┘ │ [説明テキスト]   │
│ ┌──┐ ┌──┐ ┌──┐ │                 │
│ │  │ │  │ │  │ │ [ステータス]     │
│ └──┘ └──┘ └──┘ │ [装備] [捨てる]  │
├──────────────────┴─────────────────┤
│ [ソート▼]            [12/50 アイテム]│
└────────────────────────────────────┘
```

### パフォーマンス: 大量アイテム
- 50個以上のアイテムリストでは **ListView** (UI Toolkitの仮想化リスト) を使用
- `makeItem` / `bindItem` コールバックで要素を再利用（アロケーション抑制）
- 全アイテムを一度にVisualElementとして生成しない

## 5. 設定画面

### 原則
- **カテゴリ分け**: 音声、グラフィック、操作、ゲーム等
- **即時反映**: 音量スライダーはドラッグ中にリアルタイム反映
- **保存/キャンセル**: 変更を適用前に確認、キャンセルで元に戻す
- **デフォルトリセット**: 「初期設定に戻す」ボタンを用意

### コントロール対応表

| 設定項目 | UI要素 | 備考 |
|---------|--------|------|
| 音量 | `Slider` | 0-100, ドラッグ中にプレビュー |
| 解像度 | `DropdownField` | 使用可能な解像度を列挙 |
| フルスクリーン | `Toggle` | |
| キーバインド | カスタム要素 | 「押してください...」入力待ち |
| 言語 | `DropdownField` | |

## 6. 共通パターン

### ローディング / スケルトン
- データ読み込み中は **スケルトン表示**（灰色のプレースホルダー矩形）
- スピナー（回転アイコン）はC#アニメーションで実装（USSだけではループ不可）

### ツールチップ
- `tooltip` UXML属性で簡易ツールチップ対応
- カスタムツールチップが必要な場合はVisualElementを動的に追加

### 通知 / トースト
- 画面上部中央に短時間表示 → 自動消滅
- 重要な情報はトーストだけに頼らない（ログにも残す等）

### フォーカスナビゲーション（パッド対応）
```csharp
// C#側でフォーカス制御
rootVisualElement.RegisterCallback<NavigationMoveEvent>(evt =>
{
    // カスタムナビゲーションロジック
});

// 画面表示時に初期フォーカスを設定
firstButton.Focus();
```

## 7. USS変数テーマ（プロジェクト共通）

全UI画面で共有する変数。新画面作成時はこのテーマを `@import` する。

```css
/* theme-variables.uss — 全画面共通 */
:root {
    /* カラー */
    --color-bg-overlay: rgba(0, 0, 0, 0.7);
    --color-bg-panel: rgba(20, 20, 30, 0.9);
    --color-bg-input: rgba(40, 40, 50, 0.8);
    --color-text-primary: rgb(240, 240, 245);
    --color-text-secondary: rgb(160, 160, 175);
    --color-text-muted: rgb(100, 100, 115);
    --color-accent: rgb(79, 195, 247);
    --color-danger: rgb(244, 67, 54);
    --color-success: rgb(76, 175, 80);
    --color-warning: rgb(255, 193, 7);
    --color-gold: rgb(255, 215, 0);

    /* HP/MP/スタミナ — 既存HUDと統一 */
    --color-hp: rgb(50, 200, 50);
    --color-mp: rgb(60, 120, 220);
    --color-stamina: rgb(220, 200, 50);
    --color-boss-hp: rgb(200, 50, 50);

    /* スペーシング */
    --spacing-xs: 4px;
    --spacing-sm: 8px;
    --spacing-md: 16px;
    --spacing-lg: 24px;
    --spacing-xl: 40px;

    /* フォントサイズ */
    --font-size-xs: 11px;
    --font-size-sm: 14px;
    --font-size-md: 18px;
    --font-size-lg: 24px;
    --font-size-xl: 32px;

    /* トランジション */
    --transition-fast: 0.15s;
    --transition-normal: 0.3s;

    /* ボーダー */
    --border-radius-sm: 3px;
    --border-radius-md: 6px;
    --border-radius-lg: 12px;
}
```
